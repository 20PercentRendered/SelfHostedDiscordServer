# ServCord
A self hostable Discord server. Currently very alpha.

## Features in gateway
 - [x] HELLO, HEARTBEAT_ACK and READY payloads
## Features in voice gateway
 - [ ] Connecting
 - [ ] Speaking
 - [ ] Voice events
## Features in RESTful API
 - [ ] Creating a server
 - [ ] Deleting a server
 - [ ] User Settings
 - [ ] Server Settings
## Extra features
 - [ ] Bundled BetterDiscord
 - [ ] Client mods (/api/vX/client/mods)
